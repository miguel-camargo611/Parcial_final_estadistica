📏 SUPUESTO 1: NORMALIDAD

Prueba de Mardia

Se rechazó la normalidad multivariada en todos los grupos ($p < 0.05$). La asimetría y curtosis técnica prohiben el uso de estadísticos paramétricos clásicos, motivando el uso de Bootstrap y Permutación.

📐 SUPUESTO 2: HOMOCEDASTICIDAD

Prueba M de Box

Estaciones: Matrices de covarianza heterogéneas ($p < 0.05$).
Biológico: Se cumplió la igualdad de matrices entre presencia/ausencia, aunque se mantuvo el enfoque robusto por desbalance.

Parte A: Auditoría Ambiental

El Veredicto de la Calidad del Aire

H₀: μ_sitio = μ_OMS  (Cumple)
H₁: μ_sitio ≠ μ_OMS  (Rechazo)

🎯 Metodología: Bootstrap (T²)

📏 P1: T² Observado
Se calcula el estadístico multivariado con los datos reales: T²ₒᵦₛ = n(x̄ - μ₀)ᵀ S⁻¹ (x̄ - μ₀)
📐 P2: Transformación (H₀)
Se centran los datos al estándar OMS para reflejar la hipótesis nula: Xᵢ* = Xᵢ - x̄ + μ₀
🔄 P3: Remuestreo (B=5,000)
Se generan múltiples muestras aleatorias con reemplazo de X* para recrear la variabilidad simulada.
📈 P4: p-valor Empírico
Se mide cuántos estadísticos simulados superan el real: p = [#(T²ᵦ ≥ T²ₒᵦₛ) + 1] / (B + 1)

Resultados: Test T² de Hotelling

Las 4 pruebas arrojaron alta significancia (p=0.0002). Rechazamos el cumplimiento en las zonas monitoreadas. Bogotá no cumple estándares OMS.

💡 Idea clave: El bootstrap simula el comportamiento del estadístico utilizando la estructura de covarianza real, siendo robusto ante la asimetría de los contaminantes.
Parte B: Zonificación Química

Análisis de Disimilitud Inter-Estacional

Planteamiento Formal de Hipótesis: H₀: μ₁ = μ₂ = μ₃ = μ₄  (Centroides Químicos Idénticos) H₁: ∃ μᵢ ≠ μⱼ  (Al menos una estación difiere significativamente)

🧪 Flujo de Inferencia: PERMANOVA

📏 Paso 1: Distancia Euclidiana
Calculamos la matriz de disimilitud entre las 798 observaciones usando la norma L2 en el espacio estandarizado: dᵢⱼ² = ||xᵢ - xⱼ||² = Σₖ (xᵢₖ - xⱼₖ)² Esto cuantifica la "distancia química" pura entre cada día de monitoreo en el espacio de 5 dimensiones.
📐 Paso 2: Partición de Variabilidad
Descomponemos la variabilidad total (SSₜ) en componentes intra-grupo (SS𝓌) y entre-grupos (SSₐ):
SSₜ = Σᵢ Σⱼ (dᵢⱼ² / 2N) SS𝓌 = Σₖ Σᵢₖ Σⱼₖ (dᵢⱼ² / 2nₖ) SSₐ = SSₜ - SS𝓌
🎲 Paso 3: Estadístico Pseudo-F
Construimos un análogo al estadístico F del ANOVA para evaluar la separación de centroides: F = [SSₐ / (g - 1)] / [SS𝓌 / (N - g)] Un valor F elevado indica que la separación entre estaciones supera el ruido interno.
🔄 Paso 4: Distribución Monte Carlo
Barajamos aleatoriamente las etiquetas (B = 5,000 veces) para construir una distribución nula empírica. Calculamos el p-valor empírico de forma conservadora: p = [#(Fₚₑᵣₘ ≥ Fₒᵦₛ) + 1] / (B + 1)
🔬 Paso 5: Decisión & Post-hoc
Rechazamos H₀ si p < α (0.05). Ante significancia global, aplicamos comparaciones por pares con Corrección de Bonferroni: α_adj = α / k Garantizando que la probabilidad de error Tipo I no se infle por múltiples contrastes.

🛡️ Robustez: Hotelling (Permutación)

🔬 Fase 1: Estadístico Observado (T²)
Calculamos la distancia multivariada entre centroides usando la muestra original: T² = [N₁N₂/(N₁+N₂)] (x̄₁-x̄₂)ᵀ Sₚ⁻¹ (x̄₁-x̄₂) Donde Sₚ es la matriz de covarianza combinada (pooled).
🎲 Fase 2: Simulación (Permutación)
Bajo H₀: μ₁ = μ₂, las etiquetas de grupo son irrelevantes. Barajamos aleatoriamente las etiquetas de estación entre todas las observaciones para simular el azar.
⛓️ Fase 3: Recálculo Iterativo
Repetimos el proceso B = 5.000 veces, calculando un nuevo estadístico T²ₚₑᵣₘ para cada muestra barajada aleatoriamente.
📊 Fase 4: Distribución Nula
Construimos la distribución empírica de todos los T²ₚₑᵣₘ. Esto representa todos los valores de "diferencia" que podrían ocurrir solo por puro azar.
📈 Fase 5: p-valor Empírico
Calculamos la proporción de valores aleatorios que superaron al observado (incluyendo ajuste conservador): p = [#(T²ₚₑᵣₘ ≥ T²ₒᵦₛ) + 1] / (B + 1) Si p < α,como sucedio en las 6 pruebas por pares, rechazamos la similitud.

Parte C: Bioindicadores & Resiliencia

Hábitat Atmosférico del Copetón

Planteamiento Formal de Hipótesis: H₀: μₚᵣₑₛₑₙcᵢₐ = μₐᵤₛₑₙcᵢₐ  (Nicho Atmosférico Idéntico) H₁: μₚᵣₑₛₑₙcᵢₐ ≠ μₐᵤₛₑₙcᵢₐ  (Presencia dependiente de la Firma Química)

Resultados Finales: Convergencia de Pruebas

No existe evidencia significativa de segregación química del hábitat para el Copetón (p > 0.05).
La especie se considera resiliente: su detección no se ve condicionada estadísticamente por la firma contaminante actual de la ciudad.

⚡ Interpretación: La adaptación evolutiva de Z. capensis le permite colonizar zonas críticas sin comprometer su presencia inmediata.

PERMANOVA: p=0.413
